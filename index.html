About

1 + 1 = 1. Prove that you are the pope.

My 8th grade geometry teacher started his introduction to logical proofs with this challenge. He had a fun Hawaiian shirt on, I was sitting next to all of my buddies, this seemed like it was going to be the best class.

If I am one, and the pope is one, then together we are one.

Okay, I like where the teacher's head is at, this could be fun. I'm all in on this proof business.

But since together, we're the same as just me, I must be the pope.

So it turns out, if you start with a flawed premise, you can convince yourself anything is true. My flawed premise for geometry was that Hawaiian shirts and alleged Bertrand Russell quotes implied that I would love geometry.

Since then I've been out to avoid as many flawed premises as I can.

So if you see any of my projects that seem bonkers, let me know! If there's a technology I haven't put down that I really need to try, let me know! This website isn't so that you'll hire me to consult, this is so that I can get some extra feedback or maybe even some extra dev help.
